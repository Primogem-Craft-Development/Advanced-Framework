# Advanced Framework
## A modern render framework for rendering ui and 3d models & shaders.
### Now implmented:
* UI Compositor
* MMD Renderer
### How to use:
Use the command
```
/advancedfmk open
```
to load the model.<br><br>
And use
```
/advancedfmk animation load
```
to load vmd file.
